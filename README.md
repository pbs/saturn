saturn
======

a tool for dealing with scheduled tasks on ECS

Features:

* show scheduled jobs
* see runs of scheduled jobs
* view logs of individual runs
* start one-off run of job

Not Currently Supported:

* live tail of logs
* scheduling jobs
* complex jobs with multiple containers
